# source-reading-notes

一个持续更新的源码解读仓库，主要用来整理前端框架与相关技术的源码学习笔记、系列文章和结构图。

目前会优先围绕 **React** 展开，后续也会逐步补充 **Vue** 以及其他值得系统梳理的源码专题。

## 这个仓库会放什么

这个仓库不是单纯存放零散 Markdown 文件的地方，而是一个按主题持续整理的源码解读仓库。  
我会尽量把每个系列写成结构稳定、可连续阅读的一组文章，重点放在：

- 帮助自己把源码主线真正梳理清楚
- 也让后来的读者更容易顺着主线往里读
- 尽量沉淀出能复用的结构图、问题拆解方式和写作规范
